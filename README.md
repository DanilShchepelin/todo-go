## Запуск проекта

```bash
  # Создать и настроить файл .env
  cp .env.example .env
```

### Собрать контейнер

```bash
  make app_build
```

### Поднять проект и следить за приложением в консоли

```bash
  make start_live
```

### После окончания работы, выключить контейнеры

```bash
  make down
```
